# Open the page

Just open the `index.html` on any browser.

## Compile stylesheet:

Install the `sass` cli tool:

```bash
npm install -g sass
```

Then use

```bash
sass --watch ./index.scss dist/index.css
```

And all the `.scss` are merged and compiled into a single `.css` file.
